Events Rules

Rules and Regulations for Coding Competition * Entry Fee: ₹100 per team.
* Team Composition: A team can have a maximum of 2 members.
* Competition Structure: The competition consists of 3 rounds.

Round 1: Quiz
* The quiz will consist of 50 multiple-choice questions (MCQs).
* Participants will have 60 minutes to complete the quiz.
* There will be no negative marking.
* Questions will be based on C, C++, Java, and Python.

Round 2: Code
* Participants can use any programming language of their choice.
* There will be 3 coding questions
* The time duration for this round is 1 hour.

Round 3: Decode
* Participants will be given a piece of code that may be written in C, C++, Java, or Python.
* The task is to analyse and debug the given code.
* The time duration for this round is 1 hour.

Event Rules & Guidelines:
1. Fair Play:
* Any form of plagiarism, cheating, or discussion with other teams will lead to disqualification.

2. Decision of Judges:
* The organizing committee/judges’ decision is final in case of disputes.

3. Internet & Resources:
* The internet is not allowed, except for referring to syntax documentation.
* Participants must write their own code without external help.

4. System Usage:
* Participants must use the pre-allocated systems/laptops provided or as per event guidelines.

5. Tie-Breaker Rule:
* If there is a tie, factors like execution time, code efficiency, or an extra problem-solving challenge will determine the winners.
